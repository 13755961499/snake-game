# 经典贪吃蛇小游戏 (Classic Snake Game)

这是一个使用纯前端技术（HTML5 Canvas, CSS3, JavaScript）实现的经典贪吃蛇游戏。

## 项目简介

该项目展示了如何使用 HTML5 Canvas 进行 2D 游戏开发，包含了游戏循环、碰撞检测、状态管理以及简单的动画效果。

## 核心功能

1.  **Canvas 渲染**：使用 HTML5 Canvas 绘制 20x20 的游戏网格，实时渲染蛇身和食物。
2.  **移动逻辑**：支持键盘方向键和 WASD 键控制。
3.  **碰撞检测**：
    *   **边界碰撞**：撞墙即游戏结束。
    *   **自身碰撞**：蛇头撞到身体任何部位即游戏结束。
4.  **成长机制**：蛇吃到食物后长度增加，分数增加。
5.  **动态难度**：随着分数的提高，游戏运行速度会逐渐加快，增加挑战性。
6.  **状态管理**：支持开始、暂停、恢复和重新开始功能。
7.  **最高分持久化**：使用 `localStorage` 记录并展示玩家的历史最高分。

## 文件结构

- `index.html`: 页面结构与游戏容器。
- `styles.css`: 游戏样式，包含响应式设计与深色主题。
- `script.js`: 游戏核心逻辑实现。

## 操作说明

| 按键 | 功能 |
| :--- | :--- |
| **方向键 / WASD** | 控制蛇的移动方向 |
| **空格键 (Space)** | 开始 / 暂停 / 恢复游戏 |
| **鼠标点击按钮** | 开始 / 重新开始游戏 |

## 运行方式

1. 确保所有文件 (`index.html`, `styles.css`, `script.js`) 位于同一目录下。
2. 直接使用浏览器打开 `index.html` 即可开始游戏。

## 技术亮点

- **防掉头逻辑**：在 JavaScript 中实现了方向缓冲，防止蛇在快速按键时发生 180 度掉头导致的非法自杀。
- **渲染细节**：蛇头带有眼睛细节，食物带有发光阴影效果。
- **性能优化**：通过 `setInterval` 与 `clearInterval` 配合实现动态帧率，模拟难度递增。
